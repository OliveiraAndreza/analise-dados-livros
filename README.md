# 📚 Análise de Dados de Livros

Projeto de análise exploratória de dados utilizando Python.

## Objetivo

Explorar dados de livros para entender padrões de avaliações e popularidade.

## Tecnologias

- Python
- Pandas
- Matplotlib
- Seaborn

## Análises realizadas

- limpeza de dados
- top livros avaliados
- autores com mais obras
- relação entre popularidade e nota

## Dataset

Goodreads Books Dataset
